# Komentář k zákonu č. 338/1992 Sb., o dani z nemovitých věcí (ZDNV)

Odborný komentář k **zákonu o dani z nemovitých věcí** — majetkové dani z pozemků a ze staveb a jednotek. Komentář pokrývá **všech 46 paragrafových jednotek** (§ 1–22 vč. písmenných) v devíti částech zákona.

- **Konsolidované znění:** k **1. říjnu 2025** (e-Sbírka, dokument `SB-1992-00338_2025-10-01`, document_id 1569327).
- **Rozsah:** ~29 tis. slov, 4 obsahové kapitoly + úvod + index.
- **Témata:** daň z pozemků, daň ze staveb a jednotek, základ a sazby, koeficienty (velikost obce, inflační, místní), správa daně, obecně závazné vyhlášky a OOP obcí.

## Jak komentář číst

- **[00-uvod.md](00-uvod.md)** — úvod: co zákon upravuje, systematika, metoda.
- **[INDEX.md](INDEX.md)** — rozcestník podle paragrafů.

## Metoda

Komentář je zpracován **výhradně z lokálních materiálů** — z úplného konsolidovaného znění zákona.
